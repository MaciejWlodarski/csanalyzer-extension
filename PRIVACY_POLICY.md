This Privacy Policy describes how CSAnalyzer Extension processes data when used with FACEIT match pages.

## Data Processed

The extension reads data related to FACEIT matches:

- Match ID
- Demo URLs
- Match metadata (e.g., map details, stats)

This data is used only to:

- Display the analysis UI on FACEIT match pages
- Enable demo upload to CSAnalyzer
- Check the status of analysis

## Data Transmission

When a user uploads a demo for analysis, the extension sends:

- Match ID
- Demo URL
- Demo download URL

to the CSAnalyzer backend services:

- https://collector.csanalyzer.gg
- https://art.csanalyzer.gg

## What We Do NOT Collect

The extension _does not collect_:

- Email addresses
- Passwords
- Financial information
- Personal messages
- Browsing history outside FACEIT

## Usage and Storage

Data is used strictly for the functionality described and is not stored outside of what the backend requires for demo processing.
